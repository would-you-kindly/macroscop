Исполняемый файл находится в папке cv_school_test\project\MacroscopTest\bin\Debug\MacroscopTest.exe

Для основного задания используется функция Crop в меню Effects. Предварительно нужно загрузить картинку командой File -> Open image и загрузить области обрезки командой File -> Load annotations. После этого команду Crop можно будет выполнить, и она предложит сохранить каждый фрагмент в отдельный файл.

Для дополнительных заданий используются команды Grey, Flip, Normalize и Noise (для 1-го, 2-го, 3-го и 4-го заданий соответсвенно). Для выполнения этих команд должна быть открыта картинка (командой File -> Open image) (при это загруженный области на результат этих функций не повлияют). Для сохранения обработанных картинок используется команда File -> Save image.
